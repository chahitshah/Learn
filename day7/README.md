
# Day 7 Project

## Overview
This project combines HTML and CSS to create a web interface for Day 7 learning.

## Files
- **day7.html** - HTML structure and markup
- **day7.css** - Styling and layout

## Usage
Open `day7.html` in your web browser to view the project.

## Features
- Responsive design with CSS styling
- Clean semantic HTML structure

## Getting Started
1. Ensure both `day7.html` and `day7.css` are in the same directory
2. Open `day7.html` in your preferred browser
